# 仙途 · 文字修仙

水墨画风文字修仙游戏。256 个剧情节点，最长 30 轮决策，6 条主线，数十种结局。

---

## 快速开始

### 浏览器版（推荐）

双击 **`run_gui.bat`** 即可自动安装依赖、启动服务器并打开浏览器。

或手动执行：

```bash
pip install flask
python server.py
```

浏览器访问 **http://127.0.0.1:5000** 开始游戏。

### 终端版

```bash
python game.py
```

或双击 `run.bat`。

## 角色创建

1. **输入名字** — 回车默认「叶尘」
2. **分配属性点** — 根骨、幸运、魅力、精神、悟性，共 100 点，每项不低于 5
3. **选择词条** — 6 个初始词条中选 1 个，获得额外属性加成

### 五项属性

| 属性 | 影响范围 |
|------|---------|
| 根骨 | 战斗、体魄、肉身 |
| 幸运 | 机缘、寻宝、陷阱 |
| 魅力 | 社交、说服、交易 |
| 精神 | 意志、心魔抵抗 |
| 悟性 | 学习、功法领悟 |

属性不足时选择会走向更艰难的路线。部分选择会**悄悄增加属性**（无提示），考验玩家对不同流派的理解。

### 初始词条

| 词条 | 加成 |
|------|------|
| 天生剑骨 | 根骨 +10 |
| 天命所归 | 幸运 +15 |
| 龙凤之姿 | 魅力 +15 |
| 心如磐石 | 精神 +15 |
| 七窍玲珑 | 悟性 +15 |
| 天道酬勤 | 五项各 +4 |

## 存档系统

- **手动存档**：随时保存，每次生成新文件，适合保留关键节点
- **自动存档**：每进入新章节自动覆盖同一文件，一局只留一个
- **删除存档**：读取存档弹窗中每个条目右侧有删除按钮
- 存档文件为 JSON 格式，保存在 `saves/` 目录
- 通关后重新开始会回到角色创建阶段

## 常见问题

**Q: 启动后页面空白？**

A: 确保 Flask 已安装，刷新页面。双击 `run_gui.bat` 会自动安装。

**Q: 端口 5000 被占用？**

A: 编辑 `server.py` 末尾，将 `5000` 改为其他端口（如 `8080`）。

**Q: 终端版闪退？**

A: 在终端中执行 `python game.py` 而非双击 `.py` 文件。或使用 `run.bat`。

**Q: 中文乱码？**

A: 使用 Windows Terminal 运行，或双击 `run.bat`（内置 UTF-8 编码设置）。
