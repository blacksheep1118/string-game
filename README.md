# 仙途 · 文字修仙

256 个剧情节点、最长 30 轮决策的修仙文字游戏。支持终端版和浏览器版两种游玩方式。

---

## 运行方式

### 浏览器版（推荐）

双击 `run_gui.bat`，自动打开浏览器进入游戏。水墨画风界面，鼠标点击操作。

或手动执行：

```bash
pip install flask
python server.py
# 浏览器访问 http://127.0.0.1:5000
```

### 终端版

```bash
python game.py
```

或双击 `run.bat`。

---

## 游戏流程

### 1. 角色创建

- 输入名字 → 分配 100 点属性（根骨/幸运/魅力/精神/悟性，每项≥5）→ 选择初始词条

### 2. 属性说明

| 属性 | 影响范围 |
|------|---------|
| 根骨 | 战斗、体魄、肉身 |
| 幸运 | 机缘、寻宝、陷阱 |
| 魅力 | 社交、说服、交易 |
| 精神 | 意志、心魔抵抗 |
| 悟性 | 学习、功法领悟 |

属性不足时部分选择会走向更艰难的路线。部分选择会悄悄增加属性（无提示）。

### 3. 初始词条

| 词条 | 加成 |
|------|------|
| 天生剑骨 | 根骨 +10 |
| 天命所归 | 幸运 +15 |
| 龙凤之姿 | 魅力 +15 |
| 心如磐石 | 精神 +15 |
| 七窍玲珑 | 悟性 +15 |
| 天道酬勤 | 五项各 +4 |

## 存档

- 游戏过程中随时保存，可同时保留多个存档
- 存档为 JSON 格式，保存在 `saves/` 目录

---

## 游戏数据

- **256 个剧情节点**，最长剧情线 30 轮决策
- 54 个隐藏属性加成选项
- 6 条主线：剑修 / 丹修 / 宗门 / 散修 / 古玉 / 商道
- 结局评价从 D 到 SS

## 常见问题

**Q: 浏览器版启动后页面空白？**

A: 确保已安装 Flask（`pip install flask`），刷新页面。

**Q: 端口被占用？**

A: 编辑 `server.py` 最后一行，将 `5000` 改为其他端口。

**Q: 双击 `game.py` 闪退？**

A: 在终端中执行 `python game.py`，或使用 `run.bat`。
