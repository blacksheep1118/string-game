# 仙途 · 文字修仙

水墨画风修仙文字游戏。256 个剧情节点，30 轮决策深度，6 条主线，46 种结局。

---

## 快速开始

### 浏览器版（推荐）

双击 **`run_gui.bat`** — 自动安装依赖、启动服务器、打开浏览器。

或手动：

```bash
pip install flask
python server.py
# 浏览器访问 http://127.0.0.1:5000
```

### 终端版

```bash
python game.py
```

或双击 `run.bat`。

### 打包为独立 EXE

双击 **`build_exe.bat`**，等待构建完成后 `dist/XianTu.exe` 即为独立可执行文件，无需安装 Python。

## 功能一览

| 功能 | 说明 |
|------|------|
| 角色创建 | 自由分配 100 点属性 + 6 选 1 初始词条 |
| 属性判定 | 隐藏需求，属性不足自动走艰难路线 |
| 隐藏加成 | 54 个选项会悄悄增加特定属性 |
| 自动存档 | 每章切换时覆盖式自动保存 |
| 手动存档 | 随时保存，每次生成新文件 |

| 古风音效 | Web Audio 生成，选项/章节/结局各有音效 |
| 章节过渡 | 水墨大字过场动画 |
| 粒子效果 | 背景浮动字符 |

## 常见问题

**Q: 浏览器版页面空白？**

A: 安装 Flask（`pip install flask`）后刷新页面。

**Q: 端口 5000 被占用？**

A: 编辑 `server.py` 末尾，将 `5000` 改为其他端口。

**Q: 打包 EXE 失败？**

A: 确保已安装 PyInstaller（`pip install pyinstaller`），保留 `/dist` 目录。
